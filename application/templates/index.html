Os temas do evento

  • Testes em Javascript

    Nesta palestra, será abordado:
    • como seu código javascript pode ser escrito para ser "testável";
    • o que é "mock" em termos de teste unitário e como executar em código client;
    • algumas ferramentas de teste e de mock mais usadas e
    • filosofia de testes em outras linguagens, como Python, trazidas para o código client.

    Palestrantes: Márcio Santana e Raphael Almeida

  • Ajax Crawling Links

    Uma melhor forma de trazer conteúdos exclusivos do site dando maior força à indexação das páginas principais do mesmo. Requisições em AJAX de parte do conteúdo passado via hash.

    Palestrante: ainda não confirmado

  • API Mashups

    Integração de diversas APIs: Google Maps + Geolocation do HTML 5, APIs de redes sociais, etc.

    Palestrante: Bruno Dulcetti

  • Padrões Javascript

    Padrões e antipadrões em Javascript em prática. Utilização da ferramenta JSHint para melhor utilização do código com convenções e sem permitir bolas de lama.

    Palestrante: Leo Balter

  • Performance em Front End - PageSpeed nota 100

    Veja o que pode ser feito para melhorarmos os resultados do Pagespeed, YSlow e Audit do Chrome. Minimização de código, utilização de cache, background sprites, etc

    Palestrante: ainda não confirmado

  • Criação de plugins em jQuery

    Passo a passo sobre como criar um plugin em jQuery, e a melhor forma de divulga-lo e conseguir interação com a comunidade, para uma criação colaborativa.

    Palestrante: ainda não confirmado

  • Mercado de Trabalho Front End

    O que o profissional deve fazer para entrar no mercado de trabalho de front end ou até se especializar cada vez mais. Quais caminhos seguir, onde e como estudar, estratégias.

    Palestrante: mesa redonda com convidados
